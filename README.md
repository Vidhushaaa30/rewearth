# Rewearth

## Overview
Rewearth is a web platform designed for intra-college clothing exchange. It enables students to list, browse, and swap clothing items within the campus community. The goal is to promote sustainable fashion, reduce textile waste, and make clothing more accessible and affordable for students.

## Features
- Users can upload and list clothing items for exchange
- Browse items shared by other students
- Secure authentication using Firebase
- Swap request functionality between users
- Responsive user interface built using Tailwind CSS
- Real-time updates using Firebase Firestore

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/rewearth.git
```

Navigate to the project directory

```bash
cd rewearth
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

The application will run on:

```
http://localhost:5173
```
